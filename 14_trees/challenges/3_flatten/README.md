## flatten.js

Реализуйте и экспортируйте по умолчанию функцию `flatten`, которая делает плоским вложенный массив.

```js
const list = [1, 2, [3, 5], [[4, 3], 2]];

// [1, 2, 3, 5, 4, 3, 2]
flatten(list);
```
